💳 Stripe Payment API Documentation

🟢 FastAPI: Port 8000 🟢 Django: Port 8002

FastAPI Endpoints

GET /health
Check if server is running
curl http://localhost:8000/health
GET /api/products
Get all courses and plans
curl http://localhost:8000/api/products
POST /api/create-checkout-session
Create Stripe checkout session
curl -X POST http://localhost:8000/api/create-checkout-session \
  -d "item_type=course" \
  -d "item_id=1" \
  -d "user_id=1" \
  -d "user_email=test@example.com"
GET /dashboard
View all products in grid
http://localhost:8000/dashboard
GET /checkout/course/1
Course checkout page
http://localhost:8000/checkout/course/1
GET /api/payment-status/{session_id}
Check payment status
curl http://localhost:8000/api/payment-status/cs_test_xxxxx
GET /api-docs
This documentation page
http://localhost:8000/api-docs

Django Endpoints

GET /admin
Django admin panel
http://localhost:8002/admin
GET /api/transactions/
Get all transactions
curl http://localhost:8002/api/transactions/
